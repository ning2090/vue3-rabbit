# vue3-rabbit

Vue3 + Vite + Element Plus + Vue Router + Pinia + sass + axios + VueUse
优化：图片懒加载

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
